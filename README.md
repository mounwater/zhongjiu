# 项目介绍
本项目是基于HTML、JS、CSS、JQuery等技术模拟实现中酒网的首页、详情页、购物车等界面与功能，项目及页面只用于学习用途，如有侵权请联系我。
项目如何跑起来？
## 项目如何跑起来？
本项目使用gulp构建，
首先拉取远程仓库代码后，
需要执行```npm install```安装依赖包，
然后执行```gulp default```即可运行起gulp server等环境
